尊敬的评委：

本系统为 AI 智能预测系统，为了确保机器学习模型环境（Python、XGBoost、Streamlit 等）的绝对兼容，我们已将完整的运行环境封装在 python\_env 文件夹中。

**一、 运行环境配置（免安装）**

本项目为解决跨平台兼容性问题，采用了“绿色独立环境封装”技术。系统已内置完整的 Python 解释器、XGBoost 模型及相关依赖库，评委老师无需配置任何环境，真正做到“开箱即用”。

**二、 极简启动步骤**

1.解压文件：请将整个项目压缩包解压到您的电脑本地。（重要提示：为确保底层 AI 模型正常加载，请务必保证解压后的文件夹路径中**不包含中文字符**）。

2.一键启动：进入解压后的项目根目录，**双击运行 VO2max心肺适能智能预测系统.exe**。**（若遇到系统权限或杀毒软件拦截，也可直接双击运行备用脚本 启动程序.bat）。**

3.开始体验：启动后会弹出一个黑色的命令行后台窗口，随后系统会自动在您的默认浏览器中打开交互页面。

三、 运行注意事项

关于后台进程：系统运行期间，请勿关闭黑色的命令行窗口，它是维持系统运转的核心进程。测试完毕后，直接关闭该窗口即可安全退出系统。

\*\*手动访问说明：\*\*极少数情况下，若您的浏览器没有自动弹开，请手动打开浏览器（推荐 Chrome 或 Edge），

在地址栏输入：**http://localhost:8501** 即可进入系统。
